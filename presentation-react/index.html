What is React?

  • free and open-source front-end JavaScript library for building user interfaces
  • based on UI components
  • maintained by Facebook (Meta)
  • not a framework!
React in general - free, based on components, maintained by Meta

History

  • created by Jordan Walke in 2013
  • React Native (Android, iOS) - 2015
  • React Fiber replaced Stack algorithm - 2017
  • Latest version - 17.0.2 (as of March, 2022)
